Terminal Toolkit 2.0 README

Instructions for use:

1. Put the file Terminal_Toolkit_2.0.sh in a place in your filesystem you will remember (i.e. Desktop).
2. Open the Terminal.
3. Use the 'cd' command to change the directory to the parent folder where the Shell script is located (eg. cd path/to/file). Remember you can use drag and drop to insert the filepath into the Terminal.
4. Now that the directory is changed, use the following command to authorize useage of the Shell script: 'chmod u+x Terminal_Toolkit_2.0.sh'.
5. Execute the Shell script using the following command: './Terminal_Toolkit_2.0.sh'
